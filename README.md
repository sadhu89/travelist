#Travelist

##Dependencies
- RVM >= 1.26.11
- Ruby ~> 2.2.2p95
- MongoDB >= 3.2.0

##Setup
    bundle install
    rails s

On another terminal initialize a MongoDB server, it will keep running at localhost:27017
    mongod
